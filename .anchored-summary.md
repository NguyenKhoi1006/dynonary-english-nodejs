- **Auth**: Single Firebase Auth flow for all roles (student/tutor/admin) â€” removed separate admin REST API login

- **Route separation**: Shared pages (sessions, messages, profile) now have explicit role guards. Student routes require no specific role (just auth). Tutor/admin routes have strict role checks
